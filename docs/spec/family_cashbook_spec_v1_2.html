Wallet & Balance System Mới — quan trọng
Mục tiêu: Trả lời "Gia đình còn bao nhiêu tiền?"
Tiền gia đình
12.500.000 đ
● Ví mặc định
MB Bank
8.200.000 đ
Ví phụ
Default: 1 ví "Tiền gia đình". Advanced mode: nhiều ví. Mỗi giao dịch gắn wallet_id. Balance được update real-time và lưu denormalized.
Quy tắc cập nhật balance
Expense → trừ balance  ·  Income → cộng balance  ·  Transfer → trừ ví A, cộng ví B  ·  Fund contribution → trừ balance

Trục 5 — Nợ (Debt)
Debt + Debt Payment
Theo dõi tiến trình trả nợ. Nhắc nhở thông minh theo due_date. Loại mặc định: 🏦 Ngân hàng  ·  👤 Cá nhân  ·  💳 Thẻ tín dụng  ·  🏢 Công ty  ·  📦 Khác

Debt Payment → trừ balance của ví tương ứng khi ghi nhận trả nợ.
Monthly Budget Mới v1.2

Bắt buộc trong TẤT CẢ bảng
couple_id — Row-level security / multi-tenant isolation

updated_at   updated_by — Audit trail

is_deleted   deleted_at — Soft delete (không xóa cứng nếu có transaction)
Denormalization (performance)
Expense lưu thêm category_name + category_icon để render feed mà không cần JOIN. Balance của Wallet được lưu và update real-time (eventual consistency với fallback fetch).

Tech Stack
Mobile
Flutter — cross-platform iOS + Android. Widget quick-add ở Phase 4.
Backend
Supabase (BaaS — nhanh cho MVP). PostgreSQL. WebSocket cho real-time sync.
AI
Claude API — Spending insights, auto-categorize, monthly report. Rule-based Phase 1 → ML Phase 2.
Infra
Firebase Auth + FCM. Cloud Storage (avatar). AWS / GCP / Supabase hosting.
